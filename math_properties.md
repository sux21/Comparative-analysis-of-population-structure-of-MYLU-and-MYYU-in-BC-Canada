# Expectation
## Definitions

Discrete random variable: $EX = \sum_{k=1}^{\infty} x_{k}p_{k}$

Continuous random variable: $EX = \int\limits_{-\infty}^{+\infty} xf(x) dx$
## Properties
1. $EC=C$, $C$ is a constant 

2. $E(X+C) = EX+C$

3. $E(CX) = CEX$

4. $E(kx+b) = kEX+b$

5. $E(X \pm Y) = EX \pm EY$

6. $\text{X and Y are independent, } E(XY) = EX \cdot EY$
